# shortenizr

> URL Shortener service

# Idea
 - Micro-service with basic UI hosted at /
 - In-mem backing store
 - Use modern but lightweight libs/tools
 - Use Node V13 with native ESM - no transpiling

# Install
```shell
npm i
```

# Usage
```shell
npm run build
npm run docker:run
```
visit http://localhost:4000

# Dev
```shell
npm start
npm test
```
